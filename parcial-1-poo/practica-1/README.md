## Objetivo
Aplicar los fundamentos de la Programación Orientada a Objetos en PHP, implementando una clase con atributos privados, constructor y métodos de acceso (getters y setters), siguiendo buenas prácticas de encapsulamiento.

## Descripción de la clase creada
Se creó la clase Usuario con:
- Atributos privados: nombre y correo
- Constructor para inicializar los datos
- Métodos getters y setters para acceder y modificar los atributos

Esto permite aplicar el principio de encapsulamiento en PHP.

## Instrucciones de ejecución
1. Colocar la carpeta practica-1 dentro de htdocs (XAMPP)
2. Iniciar Apache
3. Abrir en el navegador:
   http://localhost/practica-1/index.php
4. Verificar que se muestren los datos del usuario correctamente
